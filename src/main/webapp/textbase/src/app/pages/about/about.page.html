Vision

There is no shortage of books on the internet. But #Textbase innovates in that its texts are pretty and very shareable. We offer public domain text for free but we also want to add payed content for contemporary creators. In all languages.

Every paragraph, subparagraph has a unique, pretty, meaningful, shareable URL. Soon, you will be able to refer any fragment, down to one single letter, by a meaningful, shareable, constant URL.

Here's an example:

https://textbase.scriptorium.ro/perrault/contes/le_petit_chaperon_rouge

You can refer to it on social media, interact around it.

We want books to be enticing, web3-ready, adapted to the modern web medium rather than the other way around.

Historically, books followed the medium, from pergaments, to volumens, to folios, to prints, to HTML documents. TEXTBASE books are modern books: they are both databases that you can query but also beautifully layed-out electronic readings with semantic links showing similar paragraphs, recommendation, socially connected with other people insteresed in same authors. And printable on demand, how about that?

The book used to rest closed. #TEXTBASE opens the book for you using all web technologies.
